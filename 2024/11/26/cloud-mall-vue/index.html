beforeRouteUpdate钩子函数可以在每次路由更新之前执行，在回调中执行需要数据更新的业务逻辑即可

5.4.3 拆分业务逻辑

基于逻辑函数拆分业务是指把同一个组件中独立的业务代码通过函数做封装处理，提升代码的可维护性。

实现步骤：

  1. 按照业务声明以 use 打头的逻辑函数
  2. 把独立的业务逻辑封装到各个函数内部
  3. 函数内部把组件中需要用到的数据或者方法return出去
  4. 在组件中调用函数把数据或者方法组合回来使用
